## 目录介绍
* assets 静态资源
* components 组件（共用组件 业务组件）
* const 枚举的常量及固定不变的数据
* pages 业务页面
* services api存放入口
* types 定义interface
* utils 工具类
* main.tsx 入口文件
* router.tsx 路由文件

## 基本脚手架配置
* eslint prettier husky
* vite react typescript react-router
* ui库 [arco-design](https://arco.design/react/docs/start)
* .d.ts申明文件 路径需在tsconfig.json中对应
